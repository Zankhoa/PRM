# Food Order App

## Overview
The **Food Order App** is a mobile application developed using **Flutter**.  
The application allows users to browse food items, add them to a cart, and place orders through a simple and intuitive interface.

This project is developed as a team assignment to practice Flutter development and teamwork.

## Features
- Browse food menu  
- View food details  
- Add items to cart  
- Place food orders  
- Simple and clean UI

- Flutter SDK
